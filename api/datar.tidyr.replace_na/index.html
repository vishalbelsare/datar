    function

    datar.tidyr.replace_na.replace_na(data, data_or_replace=None, replace=None)

    </>

    Replace NA with a value

    This function can be also used not as a verb. As a function called as an argument in a verb, data is passed implicitly. Then one could pass data_or_replace as the data to replace.

    Parameters
    • data (iterable of any) — The data piped in
    • data_or_replace (any, optional) — When called as argument of a verb, this is thedata to replace. Otherwise this is the replacement.
    • replace (any, optional) — The value to replace withCan only be a scalar or dict for data frame. So replace NA with a list is not supported yet.
    Returns (any)

    Corresponding data with NAs replaced
