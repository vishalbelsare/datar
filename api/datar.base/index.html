    Factors, implemented with pandas' Categorical

    Functions

    • droplevels(x) (Categorical) — drop unused levels from a factor</>
    • factor(x, levels, exclude, ordered) (Categorical) — encode a vector as a factor (the terms ‘category’ and ‘enumerated type’are also used for factors). </>
    • is_categorical(x) (bool) — Check if x is categorical data</>
    • levels(x) (Series, , tuple, or ndarray) — Get levels from a factor</>
    • nlevels(x) (int) — Get the number of levels of a factor</>
